Getting Started

Welcome to Paylio - Reactjs + Nextjs Template We would like to thank you for choosing our template - Paylio.

It’s built with Reactjs + Nextjs It’s fully responsive and looks well. It is a modern design and high-speed performance React template.

Requirements

There are system requirements in order to install and setup this template and its components properly. Make sure that you are running the node.js & npm install.

Recommended node and npm 

                    node install for your pc
                        
                            npm install

Install Npm Packages

NPM : You must have to install packages. You can do this by running npm install command from the root of your project to install all the necessary dependencies. 

                        
                            npm install

Development server

Run npm run dev for a dev server. Navigate to http://localhost:3000/ . The app will automatically reload if you change any of the source files. 

                        
                            npm run dev
